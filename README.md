# Daily Doodle Board

A simple browser-based drawing board where users can doodle using their mouse.

## Live Project Idea

This project is intentionally small and fun, built to practice:
- DOM event handling
- Canvas drawing logic
- Basic UI controls with vanilla JavaScript
- Clean project documentation for portfolio use

## Features

- Draw on an HTML `<canvas>` using mouse input
- Smooth lines while clicking + dragging
- Brush color picker
- Adjustable brush size slider
- Clear board button
- No backend and no save state (drawing resets on refresh)

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Canvas API

## Project Structure

- `index.html` - page structure and controls
- `style.css` - layout and styling
- `script.js` - drawing behavior and event handling

## How to Run Locally

1. Clone this repository:
   - `git clone https://github.com/<your-username>/Daily-Doodle-Board.git`
2. Open the project folder.
3. Open `index.html` in your browser.

No installation or build tools required.

## Deploy with GitHub Pages

1. Push this project to a GitHub repository.
2. Open your repository on GitHub.
3. Go to **Settings** -> **Pages**.
4. Under **Build and deployment**:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main`
   - **Folder**: `/ (root)`
5. Click **Save**.
6. Wait around 1-2 minutes, then refresh the Pages section.
7. Your site will be live at:
   - `https://<your-username>.github.io/Daily-Doodle-Board/`
